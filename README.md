# AetherAI Chatbot

A premium, next-generation AI assistant that supports both text generation and AI image creation. Built with HTML, CSS, and Vanilla JavaScript.

## Features

- **Text Generation**: Powered by OpenRouter (Gemini 2.0 Flash).
- **Image Generation**: Powered by Hugging Face (Stable Diffusion XL).
- **Premium UI**: Glassmorphic dark theme, smooth animations, and responsive layout.
- **Persistence**: Chat history and API configurations saved in `localStorage`.
- **Customization**: Dark/Light mode toggle.

## Getting Started

1. Open `index.html` in your browser.
2. Click the **Settings (Cog Icon)** in the top right.
3. Enter your **OpenRouter** and **Hugging Face** API keys.
4. Start chatting!

### Default Keys
This project uses placeholders for API keys to ensure security. You can provide your own keys in the application settings.
